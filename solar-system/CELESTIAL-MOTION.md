# 太阳系模拟器 - 天体运动修正说明

## 修正概述

对太阳系模拟器中的天体运动进行了科学修正，使自转和公转速度更符合真实的天体物理学规律。

## 主要修正内容

### 1. 行星自转速度修正

#### 修正前问题
- 所有行星使用相同的自转速度 `0.01`
- 没有考虑各行星实际的自转周期差异
- 金星的逆向自转没有体现

#### 修正后的自转速度
| 行星 | 自转速度 | 实际自转周期 | 说明 |
|------|----------|--------------|------|
| 水星 | 0.002 | 58.6天 | 缓慢自转 |
| 金星 | -0.0008 | 243天 | **逆向自转** |
| 地球 | 0.25 | 24小时 | 标准参考 |
| 火星 | 0.24 | 24.6小时 | 与地球相近 |
| 木星 | 0.6 | 9.9小时 | 快速自转 |
| 土星 | 0.55 | 10.7小时 | 快速自转 |
| 天王星 | 0.35 | 17.2小时 | 中等自转 |
| 海王星 | 0.38 | 16.1小时 | 中等自转 |

### 2. 月球公转速度修正

#### 修正前问题
- 月球公转速度 `0.1` 过快
- 月球绕地球转得比地球自转还快，不符合实际

#### 修正后
- **月球公转速度**: `0.037`
- **实际周期**: 27.3天绕地球一周
- **视觉效果**: 月球公转速度适中，符合观测

### 3. 卫星运动优化

#### 潮汐锁定效应
- 大多数卫星都被潮汐锁定
- 卫星自转周期 = 公转周期
- 始终以同一面朝向母行星

#### 实现方式
```javascript
// 卫星自转与公转同步（潮汐锁定）
moon.mesh.rotation.y += moon.speed * speedMultiplier * 0.5;
```

## 科学依据

### 自转速度计算基准
以地球为基准（24小时 = 1个地球日）：
- 地球自转速度设为 `0.25`
- 其他行星按比例调整

### 公转速度保持
- 保持原有的公转速度设置
- 确保行星间的相对位置关系正确
- 维持太阳系的整体运动节奏

### 特殊情况处理

#### 金星逆向自转
- 金星是太阳系中唯一逆向自转的行星
- 使用负值 `-0.0008` 表示逆向
- 视觉上可以观察到与其他行星相反的自转方向

#### 气态巨行星快速自转
- 木星、土星自转很快（约10小时）
- 设置较高的自转速度值
- 体现气态行星的特征

## 视觉效果改进

### 1. 更真实的昼夜变化
- 地球快速自转，明显的昼夜交替
- 水星缓慢自转，长时间的昼夜

### 2. 行星特征体现
- 木星快速自转，展现气态巨行星特征
- 金星逆向自转，独特的运动方式

### 3. 月球运动合理化
- 月球公转周期约为地球自转的27倍
- 符合实际观测的月相变化周期

## 性能考虑

### 计算优化
- 自转和公转分别计算
- 避免不必要的重复计算
- 保持流畅的动画效果

### 速度控制
- 所有运动都响应速度倍数控制
- 用户可以调整观察不同时间尺度的运动
- 暂停功能正常工作
