# 快速开始 - AI 进化训练

## 🚀 一分钟上手

### 1. 打开游戏
直接在浏览器中打开 `index.html`

### 2. 启动进化训练
- 找到右侧面板的 **AI MODE** 选项
- 选择 **EVOLVE**
- 训练自动开始！

### 3. 观察进化
底部会显示进化统计面板，实时更新：
- 当前代数
- 评估进度
- 最佳适应度
- 最佳权重参数

### 4. 等待训练完成
- 默认训练 100 代
- 约 2-3 小时
- 可随时点击 STOP 停止

### 5. 查看结果
训练完成后会弹出提示，显示最终结果。

---

## 📊 如何判断 AI 是否在学习？

### 观察指标：

1. **BEST FITNESS 持续上升**
   - 第 0 代：约 3000-5000
   - 第 10 代：约 8000-12000
   - 第 50 代：约 15000-25000
   - 第 100 代：约 20000-40000

2. **BEST WEIGHTS 参数变化**
   - 初始：随机值
   - 训练后：收敛到最优值
   - 通常 LINES 权重会增大
   - HOLES 权重会变得更负

3. **游戏表现提升**
   - 初期：经常快速死亡
   - 中期：能消除一些行
   - 后期：能稳定玩很久

## 🎯 训练技巧

### 快速训练（测试用）
修改 `genetic-trainer.js` 中的参数：
```javascript
this.populationSize = 10; // 减少种群大小
this.gamesPerIndividual = 2; // 减少每个体游戏数
```
然后训练 20 代，约 20 分钟完成。

### 深度训练（最佳效果）
使用默认参数：
```javascript
this.populationSize = 20;
this.gamesPerIndividual = 3;
```
训练 100-200 代，约 4-6 小时。

### 超级训练（极致优化）
增大参数：
```javascript
this.populationSize = 50;
this.gamesPerIndividual = 5;
```
训练 200+ 代，需要一整天。

## ⚙️ 高级设置

### 调整变异率
在 `genetic-trainer.js` 中：
```javascript
this.mutationRate = 0.1; // 10% 概率变异
this.mutationStrength = 0.2; // 变异强度 20%
```

- 变异率太低：进化慢，容易陷入局部最优
- 变异率太高：不稳定，难以收敛

### 调整选择压力
修改锦标赛大小：
```javascript
tournamentSelection() {
    const tournamentSize = 3; // 增大 = 选择压力更大
    // ...
}
```

## 🐛 常见问题

### Q: 训练很久但适应度不提升？
A: 可能陷入局部最优，点击 RESET 重新开始。

### Q: 训练中途关闭浏览器怎么办？
A: 训练进度自动保存，重新打开选择 EVOLVE 继续训练。

### Q: 如何使用训练好的 AI？
A: 训练完成后，选择 AI MODE = ON，AI 会使用最佳权重自动玩。

### Q: 可以导出训练结果吗？
A: 打开浏览器控制台，输入：
```javascript
localStorage.getItem('tetris_genetic_progress')
```
复制结果保存即可。
