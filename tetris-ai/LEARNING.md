# AI 学习机制说明

## 当前状态：不会学习 ❌

目前实现的是**固定参数启发式算法**：
- 权重参数是写死的（0.76, -0.51, -0.35, -0.18）
- 每次决策都用相同的评分函数
- 训练统计只是记录表现，不影响 AI 决策
- **AI 不会变聪明**

## 让 AI 学习的三种方案

### 方案 1：遗传算法 ⭐ 推荐
**原理：** 模拟生物进化，优化权重参数

**流程：**
```
1. 初始化 20 个随机参数组合（种群）
2. 每个参数玩 5 局游戏，计算平均分（适应度）
3. 选择表现好的参数（选择）
4. 组合优秀参数生成新参数（交叉）
5. 随机调整参数（变异）
6. 重复 2-5 步，不断进化
```

**优点：**
- 实现简单（已提供 genetic-trainer.js）
- 训练快速（几小时即可）
- 效果显著（可达数万分）
- 不需要深度学习框架

**训练时间：** 2-4 小时
**预期提升：** 平均分从 5000 提升到 20000+

---

### 方案 2：强化学习（RL）
**原理：** AI 通过试错学习最优策略

**算法选择：**
- DQN（Deep Q-Network）- 最简单
- PPO（Proximal Policy Optimization）- 更稳定
- A3C（Asynchronous Actor-Critic）- 并行训练

**流程：**
```
1. AI 玩游戏
2. 记录 (状态, 动作, 奖励)
3. 用神经网络学习 状态→动作 的映射
4. 不断优化网络参数
```

**优点：**
- 理论上限最高
- 可以发现人类想不到的策略

**缺点：**
- 实现复杂（需要 TensorFlow.js）
- 训练慢（需要数万局游戏）
- 不一定比启发式好

**训练时间：** 数天到数周
**预期提升：** 不确定，可能不如遗传算法

---

### 方案 3：搜索算法
**原理：** 模拟未来 3-5 个方块，搜索最优路径

**算法：**
- Beam Search（束搜索）
- Monte Carlo Tree Search（蒙特卡洛树搜索）

**优点：**
- 效果最强（可达数十万分）
- 不需要训练

**缺点：**
- 计算量大
- 实现复杂

---

## 推荐实现路径

### 阶段 1：遗传算法（当前可实现）✅
1. 使用提供的 `genetic-trainer.js`
2. 训练 20 代（约 2-4 小时）
3. 获得优化的权重参数
4. AI 平均分提升 3-5 倍

### 阶段 2：添加更多特征
优化评分函数，增加新特征：
- 井深度（深坑惩罚）
- 行过渡（表面粗糙度）
- 列高度方差
- 可达性（是否能填补空洞）

### 阶段 3：强化学习（可选）
如果遗传算法效果不够好，再尝试 RL。

## 如何集成遗传算法

### 1. 引入训练器
```html
<!-- 在 index.html 中添加 -->
<script src="genetic-trainer.js"></script>
```

### 2. 修改 Game 类
```javascript
constructor() {
    // ...
    this.geneticTrainer = new GeneticTrainer();
    
    // 使用进化的权重
    if (this.geneticTrainer.bestWeights) {
        this.ai.weights = this.geneticTrainer.getBestWeights();
    }
}
```

### 3. 添加训练模式
```javascript
// 开启进化训练
async startEvolutionTraining(generations = 10) {
    this.geneticTrainer.initializePopulation();
    
    for (let i = 0; i < generations; i++) {
        await this.geneticTrainer.evolveGeneration();
    }
    
    // 应用最佳权重
    this.ai.weights = this.geneticTrainer.getBestWeights();
}
```

### 4. UI 控制
添加按钮：
- "开始进化训练"
- "停止训练"
- "应用最佳权重"
- "重置训练"
