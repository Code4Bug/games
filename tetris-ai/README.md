# Tetris AI - 俄罗斯方块 AI 训练系统

基于启发式算法的俄罗斯方块 AI 自动玩游戏系统。

## 功能特性

### 1. 启发式 AI 算法
采用经典论文参数的评分系统：
- **完整行数** (权重: 0.76) - 优先消除行
- **总高度** (权重: -0.51) - 保持棋盘低矮
- **空洞数量** (权重: -0.35) - 避免产生空洞
- **凹凸度** (权重: -0.18) - 保持表面平整

### 2. AI 训练模式
- 自动玩游戏
- 实时统计训练数据
- 游戏结束后自动重新开始
- 记录平均分数、平均行数、最佳分数

### 3. 训练统计
- **游戏局数** - 已完成的游戏总数
- **平均分数** - 所有游戏的平均得分
- **平均行数** - 所有游戏的平均消除行数
- **最佳分数** - 历史最高分

## 使用方法

### 启动 AI 模式
1. 打开游戏页面
2. 在右侧面板找到 "AI MODE" 选项
3. 选择 "ON" 开启 AI 模式
4. 点击 "START" 开始训练

### 查看训练统计
AI 模式开启后，底部会显示训练统计面板，实时更新：
- 游戏局数
- 平均表现
- 最佳成绩

### 重置统计
点击统计面板下方的 "RESET STATS" 按钮可以清空所有训练数据。

## AI 算法原理

### 评分函数
```javascript
score = 0.76 * completeLines 
      - 0.51 * aggregateHeight 
      - 0.35 * holes 
      - 0.18 * bumpiness
```

### 决策流程
1. **枚举所有可能的落点** - 遍历所有旋转状态和水平位置
2. **模拟方块放置** - 对每个可能位置进行模拟
3. **计算评分** - 使用启发式函数评估棋盘状态
4. **选择最佳移动** - 选择评分最高的位置
5. **执行移动** - 旋转、移动、硬降

## 技术实现

### 核心类
- `TetrisAI` - AI 评分和决策系统
- `Game` - 游戏主逻辑，集成 AI 控制
- `Board` - 棋盘状态管理
- `Tetromino` - 方块类型和旋转

### AI 特性
- 实时计算最佳移动
- 可调节移动延迟（默认 50ms）
- 自动重新开始训练
- 持久化统计数据

## 性能表现

启发式 AI 通常可以达到：
- 平均分数：数千到数万分
- 平均行数：数十到数百行
- 稳定性：高度稳定，很少出现低分

## 使用方法

### 模式 1：手动玩游戏
1. AI MODE 选择 "OFF"
2. 点击 START 开始游戏
3. 使用键盘控制

### 模式 2：AI 自动玩（固定参数）
1. AI MODE 选择 "ON"
2. 点击 START
3. AI 使用固定参数自动玩
4. 查看训练统计面板

### 模式 3：进化训练 ⭐ 让 AI 学习
1. AI MODE 选择 "EVOLVE"
2. 自动开始进化训练
3. 观察进化统计面板：
   - **GENERATION** - 当前代数
   - **INDIVIDUAL** - 当前评估的个体
   - **BEST FITNESS** - 历史最佳适应度
   - **AVG FITNESS** - 当前代平均适应度
   - **BEST WEIGHTS** - 最佳权重参数

### 进化训练说明

**训练过程：**
- 每代有 20 个个体（不同的参数组合）
- 每个个体玩 3 局游戏
- 计算平均表现作为适应度
- 选择优秀个体，交叉变异生成下一代
- 不断进化，参数越来越优

**训练时间：**
- 默认训练 100 代
- 每代约 1-2 分钟
- 总计约 2-3 小时

### 控制按钮：
- **STOP** - 停止训练
- **RESET** - 重置训练进度
- **EXPORT** - 导出训练模型（JSON 文件）
- **IMPORT** - 导入训练模型

**训练进度自动保存**，刷新页面后可继续训练。

## 💾 持久化机制

### 自动保存 ✅
训练进度会自动保存到浏览器 localStorage：
- 每代训练完成后自动保存
- 发现更好的权重时自动保存
- 包含：代数、最佳权重、适应度、整个种群

### 自动加载 ✅
每次打开页面时自动恢复上次的训练状态：
- 关闭浏览器后重新打开 → 继续训练
- 刷新页面 → 继续训练
- 切换到 AI MODE = ON → 自动使用最佳权重

### 导出/导入 ✅
- **EXPORT** - 下载训练模型为 JSON 文件
- **IMPORT** - 从文件导入训练模型
- 可以备份、分享、跨设备使用

### 使用场景
```
场景 1：长时间训练
训练 50 代 → 关闭浏览器休息
第二天打开 → 选择 EVOLVE → 从第 51 代继续 ✅

场景 2：备份模型
训练完成 → 点击 EXPORT → 保存 JSON 文件
清理浏览器缓存 → 点击 IMPORT → 恢复模型 ✅

场景 3：分享模型
训练出强大 AI → 导出文件 → 发给朋友
```

## ⚠️ 重要说明：AI 不会学习

**当前实现使用固定参数的启发式算法，AI 不会通过训练变聪明。**

训练统计只是记录表现数据，不影响 AI 的决策。每次游戏 AI 都使用相同的评分函数和权重参数。

### 如何让 AI 学习？

查看 `LEARNING.md` 了解三种让 AI 越来越聪明的方案：
1. **遗传算法** ⭐ 推荐 - 2-4小时训练，平均分提升3-5倍
2. **强化学习** - 需要数天训练，效果不确定
3. **搜索算法** - 最强但最复杂

已提供 `genetic-trainer.js` 遗传算法框架，可直接集成使用。
