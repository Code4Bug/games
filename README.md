# 🎮 经典游戏合集

一个包含多款经典游戏的 Web 应用集合，使用纯 HTML、CSS 和 JavaScript 开发，无需任何框架或依赖。

> ⚠️ **持续迭代中** - 项目正在积极开发，功能和界面可能会持续优化和更新。

## 📦 项目结构

```
.
├── Maze/              # 迷宫探险游戏
├── Minesweeper/       # 扫雷游戏
├── math-game/         # 数学小天才（儿童数学游戏）
├── tetris/            # 俄罗斯方块
└── docs/              # 项目文档
```

## 🎯 游戏列表

### 1. 迷宫探险 (Maze)
- **快速开始**: [点击进入游戏](Maze/index.html)
- **特点**: 
  - 多种难度选择（简单/普通/困难/自定义）
  - 自动寻路提示功能
  - 最佳成绩记录
  - 精美的动画效果
- **操作**: 使用方向键控制移动

### 2. 扫雷游戏 (Minesweeper)
- **快速开始**: [点击进入游戏](Minesweeper/index.html)
- **特点**:
  - 三种预设难度（初级/中等/高级）
  - 支持自定义棋盘大小和地雷数量
  - 双击数字自动展开功能
  - 最佳成绩统计
- **操作**: 
  - 左键点击打开方格
  - 右键点击标记地雷

### 3. 数学小天才 (Math Game)
- **快速开始**: [点击进入游戏](math-game/index.html)
- **特点**:
  - 适合儿童的可爱界面设计
  - 多种题型（加减法/乘法/混合）
  - 难度范围可调（10/20/50/100以内）
  - 两种做题模式（卡片模式/试卷模式）
  - 内置加法和乘法口诀表
  - 历史记录功能
- **目标用户**: 学龄儿童数学练习

### 4. 俄罗斯方块 (Tetris)
- **快速开始**: [点击进入游戏](tetris/index.html)
- **特点**:
  - 经典像素风格设计
  - 多种难度等级（简单/普通/困难）
  - 可调节棋盘宽度（8/10/12/16列）
  - 两种游戏模式（基础/高级）
  - Hold 功能（暂存方块）
  - 最佳成绩记录
- **操作**:
  - ⬅ ➡ 移动
  - ⬆ 旋转
  - ⬇ 加速下落
  - SPACE 瞬间下落
  - C 暂存方块

## 🚀 快速开始

### 方式一：直接打开
1. 克隆或下载本项目
2. 进入对应游戏目录
3. 双击 `index.html` 文件在浏览器中打开

### 方式二：本地服务器
```bash
# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx http-server
```

然后访问：
- 迷宫探险: http://localhost:8000/Maze/
- 扫雷游戏: http://localhost:8000/Minesweeper/
- 数学小天才: http://localhost:8000/math-game/
- 俄罗斯方块: http://localhost:8000/tetris/

## 💻 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式和动画效果
- **JavaScript (ES6+)**: 游戏逻辑
- **Canvas API**: 图形渲染（迷宫、俄罗斯方块）
- **LocalStorage**: 本地数据存储

## 🎨 设计特点

- 响应式设计，适配不同屏幕尺寸
- 现代化的 UI 界面

- `index.html` - 页面结构
- `styles.css` - 样式定义
- `main.js` - 游戏逻辑
